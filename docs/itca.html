compute_itca(X, y, clf, mapping, kfolds=5, y_dist=None, return_class_acc=False)¶

Given the data and a classifier, compute ITCA through cross validation.

Parameters

  • X (array like of shape (n_samples, n_features)) – The training input samples.

  • y (array like of shape (n_samples, )) – The target class labels. Integers from 0 to K-1.

  • clf (scikit-learn like classifier) – Classifiers that implements the standard interfaces of scikit-learn, including fit(X, y) and predict(X).

  • kfolds (int or sklearn.model_selection.KFold instance) – Folds of cross validation.

  • y_dist (dict, optional) – The classes distribution of y. If y_dist is None, this method will compute it automatically.

  • return_class_acc (bool) – Return the class-wise accuracy.

Returns

  • itca_cv ((kfolds, ) ndarray) – The cross validated ITCA.ndarray of ITCA.

  • class_acc (dict of accuracy of each class.) – Accruacy of each class.

Raises

  • ValueError – kf is illegal.

  • ValueError – y_dist is illegal.
